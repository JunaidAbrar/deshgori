# W1 — টাকা দেওয়ার আগে যাচাই করুন

Flagship worker-track deck (Session A). This markdown file is the **single source
of truth** for the W1 cards and quizzes. The human-readable notes live here; the
machine-readable deck is the fenced `json` block at the bottom — the app parses
that block at build time (`app/lib/content.ts`).

## Editing rules (do not "improve away")

- Card body: **≤ 2 short Bangla sentences.** Icons carry meaning alongside text.
- Any helpline number, fee, government cost ceiling, portal URL, or legal claim
  that is not yet vetted stays written as `[VERIFY: …]` until Session C clears it.
  `oc.bmet.gov.bd` is the one portal named as fact in CLAUDE.md, so it may ship.
- Quiz: 1 question, 3 options, retriable. A wrong answer shows a one-line
  explanation, then the learner retries. A correct answer advances.

## Content coverage (from the curriculum)

1. The "free visa" myth — a work visa is always tied to a specific employer + job.
2. Checking an agency's RL licence — `oep.gov.bd` → Agent List (status must read
   Active with a future validity date). Verified via Fable research (Part A).
3. Checking your own BMET status on `oc.bmet.gov.bd`.
4. Saudi visa verification — `visa.mofa.gov.sa` (free; Enjaz is retired). Verified
   via Fable research (Part A).
